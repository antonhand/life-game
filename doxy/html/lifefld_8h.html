Заголовочный файл структур и функций для работы с полем

Данный файл содержит в себе определения структур и функций, необходимых для разделения поля между процессами, создания и удаления участков поля данных процессов.

int *  y,
struct fld_param *  fldp,
struct proc_param *  proc 
)

Проверяет, пренадлежит ли клетка с координатами (x, y) участку поля данного процесса. В случае если y равен NULL, проверяет, принадлежит ли часть строки с номером x участку поля данного процесса. Приводит х и y к внутренним координатам процесса.

Аргументы
[in,out]xНомер клетки по вертикали
[in,out]yНомер клетки по горизонтали
[in]fldpПараметры поля
[in]procПараметры процесса
